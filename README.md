# Python Banking System

A simple banking system built using Python and SQLite.  
This project allows users to create accounts, login, deposit money, withdraw money, and check account balance.

## Features
- User Registration
- Secure Login
- Deposit Money
- Withdraw Money
- Check Account Balance

## Technologies Used
- Python
- SQLite
- Visual Studio Code

## Project Structure
Banking-system-python
│
├── bank.py
├── README.md
└── .gitignore

## How to Run the Project

1. Install Python on your system
2. Download or clone this repository
3. Open the project in Visual Studio Code
4. Run the following command in the terminal:

python bank.py
